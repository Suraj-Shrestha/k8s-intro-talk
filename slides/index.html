What is Kubernetes (k8s)?

  • Invented at Google, based on ideas from Borg
  • Container orchestration
  • Infrastructure component

What problems do k8s solve?

Solves many issues usually solved manually or by in-house tools and scripts

Handle that a correct number of instances of a service is always available

Even when a Node in your Cluster suddenly breaks or is unavailable

Gradual rollouts - No downtime

Security: Limiting access to containers outside the cluster

k8s concepts: Nodes

Physical machine or virtual machine in your cluster

k8s concepts: Pods

  • Lowest unit in k8s
  • Contain one or more containers (usually just one)

k8s concepts: Deployments

  • ReplicaSets
  • Main resource for content in cluster

k8s concepts: Deployments II: YAML config

							
apiVersion: apps/v1
kind: Deployment
metadata:
  name: bookservice-deployment
  labels:
    app: bookservice
spec:
  replicas: 2
  selector:
    matchLabels:
      app: bookservice
  template:
    metadata:
      labels:
        app: bookservice
    spec:
      containers:
        - name: bookservice
          image: themkat/k8sintro-bookservice:1.0
          imagePullPolicy: IfNotPresent
          ports:
            - containerPort: 8080

k8s concepts: Services

k8s concepts: Services II: YAML config

							
apiVersion: v1
kind: Service
metadata:
  name: bookservice-service
spec:
  selector:
    app: bookservice
  type: ClusterIP
  ports:
    - protocol: TCP
      port: 8080
      targetPort: 8080
